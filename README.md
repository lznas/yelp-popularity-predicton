# Yelp Restaurant Popularity Prediction & Sentiment Analysis

## Overview
This project analyzes Yelp data from Philadelphia to predict restaurant popularity using machine learning and sentiment analysis.

The goal is to understand what factors (e.g., reviews, ratings, sentiment, and business attributes) contribute most to a restaurant’s success.

---

## Dataset
- Yelp Open Dataset (filtered to Philadelphia)
- Includes:
  - Business data
  - Aggregated review statistics
  - Sample of 5,000 reviews for text analysis

! Note: Raw data is not included in this repository. It is automatically downloaded using the setup code.
